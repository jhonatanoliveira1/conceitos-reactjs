# Reactjs concepts
Challenge applied in the GoStack Bootcamp

## Requirement

* [Node](https://nodejs.org/en/)
* [Yarn](https://yarnpkg.com/)

## Installation

Clone repository
```bash
git clone https://github.com/jhonatanoliveira1/conceitos-reactjs.git
```
Install node module folder
```bash
yarn
# yarn install
```

## Usage

Sart testing
```bash
yarn test
#yarn test --watchAll 
```
